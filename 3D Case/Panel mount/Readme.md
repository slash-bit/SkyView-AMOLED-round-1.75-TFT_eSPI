These STL files are the basic designs to hold this display and mount it to standard instrument panel cutouts.

They should be 3d printable on most consumer 3d printers and have designed in bridging tabs for certain elements like the push buttons.

The designs are freely available via Onshape for anyone wishing to make changes to them.
