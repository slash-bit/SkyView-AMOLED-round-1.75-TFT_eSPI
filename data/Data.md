# Data

### OGN database
- Download the latest version of the OGN database from [this location](http://soaringweather.no-ip.info/ADB/data/ogn.cdb)
- Replace the **ogn.cdb** file in the **/Aircrafts** folder on the SD card.
- Verify on the **Status** page of the SkyView Web Server at [http://192.168.1.1](http://192.168.1.1) if the database loaded correctly.
